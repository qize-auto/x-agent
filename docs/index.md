# X-Agent

通用 AI Agent 框架，支持多模型 API、工具调用、智能路由、GUI + CLI + VS Code 三模式。

## 核心特性

- **多模型支持**: OpenRouter / OpenAI / Anthropic / Moonshot(Kimi) / DeepSeek / Ollama
- **智能模型路由**: 根据任务类型自动选择最优模型
- **视觉感知**: 截图 OCR / 无障碍树 / 多模态模型，UI → 代码关联
- **工作流引擎**: YAML 定义图结构工作流，支持条件分支和并行执行
- **SWE-bench 评估**: 完整的评估流水线

## 安装

```bash
pip install xagent
```
